Валюты

Backend веб-приложение на Python с MVC архитектурой

ОПИСАНИЕ

В этом проекте я разработал backend веб-приложение на Python с использованием архитектуры MVC. Приложение реализует полный набор CRUD-операций для управления пользователями и валютами. Проект включает собственный HTTP-сервер, систему маршрутизации и шаблоны для отображения данных.

ОСНОВНЫЕ ВОЗМОЖНОСТИ

  • создание, просмотр, обновление и удаление пользователей
  • работа с валютами и связями между пользователями и валютами
  • HTTP-маршрутизация
  • отображение данных через Jinja2 шаблоны
  • хранение данных в SQLite базе данных
  • модульная архитектура проекта

ИСПОЛЬЗОВАННЫЕ ТЕХНОЛОГИИ

  • Python
  • SQLite (in-memory database)
  • Jinja2
  • MVC архитектура
  • unittest и mock
  • HTTP server

АРХИТЕКТУРА

Проект разделен на логические слои:

  • models — модели данных
  • controllers — бизнес-логика и CRUD операции
  • templates — HTML шаблоны
