# ВЕБ-ПРИЛОЖЕНИЕ ДЛЯ УПРАВЛЕНИЯ ЗАМЕТКАМИ НА DJANGO

## ОПИСАНИЕ

Это веб-приложение для создания и управления заметками, разработанное на Python с использованием Django. Пользователь может зарегистрироваться, войти в систему, создавать заметки, редактировать их и удалять.
Проект реализован как полноценное backend-приложение с базой данных, системой аутентификации и административной панелью.

## ОСНОВНЫЕ ВОЗМОЖНОСТИ

- регистрация и авторизация пользователей
- создание, просмотр, редактирование и удаление заметок
- добавление тегов к заметкам
- поиск и организация заметок
- административная панель Django
- работа с базой данных SQLite

## ИСПОЛЬЗОВАННЫЕ ТЕХНОЛОГИИ

- Python
- Django
- SQLite
- HTML
- Bootstrap
- Django ORM

## АРХИТЕКТУРА

Проект построен по архитектуре Django (MTV):

- Model — модели заметок, тегов и профиля пользователя
- View — логика обработки запросов и CRUD операций
- Template — HTML шаблоны интерфейса
- Admin panel — управление данными через админ-панель
