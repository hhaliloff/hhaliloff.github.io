# ЧИСЛЕННЫЕ ВЫЧИСЛЕНИЯ И АНАЛИЗ ДАННЫХ С NUMPY

## ОПИСАНИЕ

В этом проекте я реализовал численные вычисления и анализ данных на Python. Проект демонстрирует использование библиотеки NumPy для работы с векторами и матрицами, выполнения математических операций и статистического анализа данных.
Также реализована загрузка данных из CSV файла и их визуализация с помощью Matplotlib и Seaborn. Проект включает набор автоматических тестов для проверки корректности реализованных функций.

## ОСНОВНЫЕ ВОЗМОЖНОСТИ

- создание и преобразование массивов NumPy
- векторные и матричные вычисления
- решение систем линейных уравнений
- статистический анализ данных
- загрузка данных из CSV файлов
- нормализация данных
- визуализация данных с помощью графиков
- сохранение графиков в отдельную папку
- автоматическое тестирование функций
