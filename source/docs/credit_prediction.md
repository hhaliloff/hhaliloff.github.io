# ПРЕДСКАЗАНИЕ ДЕФОЛТА ПО КРЕДИТУ

## ОПИСАНИЕ

В этом проекте я исследовал задачу классификации клиентов банка по вероятности дефолта по кредиту. В качестве исходных данных использовался набор данных о заемщиках, содержащий финансовые и демографические признаки клиентов.

Цель работы заключалась в построении модели машинного обучения, которая может предсказывать наличие просрочки по кредиту более 90 дней. Для этого были выполнены загрузка данных, предварительная обработка признаков, обучение моделей классификации и оценка их качества.

Особое внимание было уделено сравнению разных алгоритмов машинного обучения. Помимо базовых моделей, были исследованы дополнительные модели градиентного бустинга CatBoost и XGBoost. Качество моделей сравнивалось по метрике F1-score, так как в задаче присутствует несбалансированность классов.

## ОСНОВНЫЕ ВОЗМОЖНОСТИ

* загрузка и анализ данных о заемщиках
* обработка пропущенных значений
* подготовка обучающей и тестовой выборок
* обучение моделей классификации
* сравнение качества моделей машинного обучения
* оценка результатов с помощью F1-score
* анализ матрицы ошибок
* работа с вероятностями принадлежности к классу
* исследование влияния порога классификации

## РЕЗУЛЬТАТЫ

В ходе работы были рассмотрены модели логистической регрессии, случайного леса, CatBoost и XGBoost.

По результатам сравнения модель CatBoost показала лучший результат среди рассмотренных моделей. Значение F1-score для CatBoost составило 0.2886, что выше результата модели Random Forest, для которой F1-score составил 0.2735.

Это показывает, что использование градиентного бустинга позволило улучшить качество классификации по сравнению с базовыми моделями.

## ВЫВОД

В результате лабораторной работы была решена задача классификации клиентов банка по вероятности дефолта. Были изучены основные этапы работы с данными: загрузка, предварительная обработка, обучение моделей и оценка качества.

Сравнение моделей показало, что CatBoost оказался наиболее эффективным алгоритмом для данной задачи. Он продемонстрировал лучшее значение F1-score, чем Random Forest, поэтому его можно считать более подходящей моделью для рассматриваемого набора данных.
