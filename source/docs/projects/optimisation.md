# Оптимизация Python-приложений: multiprocessing, threading и Cython

## ОПИСАНИЕ

Этот проект посвящен исследованию производительности Python и реализации различных методов оптимизации вычислений.
Я реализовал несколько версий одной вычислительной задачи и сравнил их производительность.

## РЕАЛИЗОВАННЫЕ МЕТОДЫ ОПТИМИЗАЦИИ

- стандартное выполнение Python
- многопоточность (threading)
- многопроцессность (multiprocessing)
- оптимизация с использованием Cython
- использование nogil для параллельных вычислений

## РЕЗУЛЬТАТЫ

В ходе тестирования было выявлено, что multiprocessing показал наилучшую производительность благодаря использованию нескольких ядер процессора.
Это позволило значительно сократить время выполнения вычислений.
