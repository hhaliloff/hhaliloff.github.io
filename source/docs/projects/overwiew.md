# ПРОЕКТЫ

## [САЙТ ОТСЛЕЖИВАЮШИЙ КУРСЫ ВАЛЮТ](projects/currency-tracker.md)

Backend веб-приложение с возможностью отслеживания курсов валют в реальном времени. Использует API для получения данных о курсах валют и предоставляет пользователю удобный интерфейс для просмотра и сравнения курсов.


## [ОПТИМИЗАЦИЯ АЛГОРИТМОВ](projects/optimization.md)

Исследование и реализация оптимизации Python-программ с использованием потоков, процессов и Cython.


## ВЕБ-ПРИЛОЖЕНИЕ ДЛЯ УПРАВЛЕНИЯ ЗАМЕТКАМИ
Это веб-приложение для создания и управления заметками, разработанное на Python с использованием Django. Пользователь может зарегистрироваться, войти в систему, создавать заметки, редактировать их и удалять.
